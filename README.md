# RTK Query Complete

A modern React application built with React, Redux Toolkit, RTK Query, React Router, and Vite.

## Features

* RTK Query for API data fetching and caching
* Redux Toolkit for state management
* React Router for navigation
* JSON Server for mock API
* Loading and error handling
* Fast development with Vite
* Responsive UI

## Tech Stack

* React 19
* Redux Toolkit
* RTK Query
* React Router DOM
* Vite
* Tailwind CSS
* JSON Server

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Run JSON Server

```bash
npm run json-server
```

## Build Project

```bash
npm run build
```
